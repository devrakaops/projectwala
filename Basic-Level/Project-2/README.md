## üìåProject Overview

In this project, we move beyond **beginner-level application deployment** and focus on deploying a **three-tier LAMP-based application**.

The application stack includes:

* **Linux** (Operating System)
* **Apache** (Web Server)
* **MySQL / MariaDB** (Database)
* **PHP / WordPress** (Application Layer)

This project is designed to help you understand how a real-world **three-tier architecture** works across different environments.
